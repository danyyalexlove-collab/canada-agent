# Agente Virtual Canadá 🍁
**Por Lolo Digital**

Agente de información sobre migración, finanzas, trámites y refugio en Canadá para latinos.

## Deploy en Vercel

1. Sube esta carpeta a un repositorio en GitHub
2. Entra a vercel.com y conecta tu GitHub
3. Importa el repositorio
4. En "Environment Variables" agrega:
   - `VITE_ANTHROPIC_API_KEY` = tu API key de Anthropic
5. Clic en Deploy

## Desarrollo local

```bash
npm install
npm run dev
```
